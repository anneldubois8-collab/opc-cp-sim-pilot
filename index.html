Pilot Case: Adult Transtibial (Dysvascular)

This is a study tool inspired by the ABC CP Written Simulation exam. It is not an official ABC product and does not use actual exam items.

You will work through a single transtibial case in several sections:

  • Patient Assessment
  • Problem Interpretation
  • Treatment Plan & Initial Implementation

For each possible action, choose whether you would perform it in this situation. After answering, you will see:

  • Core exam-style classification (Essential / Appropriate / Low value / Inappropriate / Contraindicated)
  • Brief core reasoning
  • Enrichment commentary with extra biomechanical / orthotic detail

Session Summary

This summary is for study purposes only. It does not represent ABC scoring, but it mirrors how simulation exams reward appropriate actions and penalize unsafe ones.
